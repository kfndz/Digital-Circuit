# Termômetro Digital Simulado

Este projeto consiste em uma **simulação de um termômetro digital**, desenvolvido para a disciplina de Sistemas Digitais.

O sistema simula o funcionamento de um dispositivo eletrônico de medição de temperatura ambiente, demonstrando as etapas básicas de leitura, processamento e exibição de dados em um sistema digital.

## Objetivo

O objetivo do projeto é demonstrar, por meio de software, como um sistema digital pode realizar a leitura de um sensor de temperatura, processar os dados e exibir o resultado em um display.

Embora o projeto utilize valores simulados, ele representa o funcionamento de um sistema real que poderia ser implementado utilizando sensores físicos e microcontroladores.

## Funcionamento do Sistema

O sistema foi dividido em etapas que simulam o funcionamento de um circuito digital:

1. **Sensor de Temperatura (Simulado)**  
   Gera um valor de temperatura que representa a leitura de um sensor analógico.

2. **Conversão Analógico-Digital (ADC)**  
   O valor analógico é convertido para um valor digital, simulando o funcionamento de um conversor ADC.

3. **Processamento do Circuito**  
   O valor digital é processado para gerar a temperatura final exibida ao usuário.

4. **Display Digital**  
   A temperatura é exibida em um display digital na interface da aplicação.

## Funcionalidades

- Simulação de sensor de temperatura
- Conversão analógico-digital simulada
- Exibição da temperatura em display digital
- Medição automática em tempo real
- Atualização da temperatura a cada 1 segundo
- Exibição do horário da última medição
- Botões para iniciar e parar as medições

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Observação

Este projeto utiliza **valores simulados de temperatura**.  
Em uma implementação real, o sistema poderia ser conectado a um sensor físico de temperatura e a um microcontrolador para realizar medições reais do ambiente.
